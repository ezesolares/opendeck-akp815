# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-12

### 🚀 Features

- Ported AKP815 support from the original plugin.
- Added dynamic dimension support (3x5).
- Implemented automatic LCD shutdown on plugin termination (sleep/shutdown).
- Made the plugin exclusive to Ajazz AKP815 (removed all other devices).
- Renamed package to `opendeck-akp815`.
